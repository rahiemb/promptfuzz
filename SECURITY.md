# Security Policy

## Supported Versions

PromptFuzz natively supports active security updates for v1.x configurations.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

PromptFuzz is a security tool, and bugs within its detectors or engine may lead to false-negatives (as observed in older architecture). 

Please report engine vulnerabilities directly via GitHub Security Advisories or to the maintainers' encrypted email blocks. Fixes will be shipped in patch releases immediately following verification.
